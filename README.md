# Loan Management Platform 🚀

## Description
The Loan Management Platform is a full-stack web application designed to manage real-world loan workflows efficiently.  
It provides secure access for both **Admins** and **Borrowers**, enabling loan creation, EMI scheduling, payment tracking, and transparent loan management.

This project focuses on scalability, security, and automation using modern web technologies.

---

## Key Features

### 🔐 Authentication
- Firebase Authentication for **Admins** and **Borrowers**
- Secure password handling using **bcrypt**

### 📅 EMI & Loan Management
- Automated EMI schedule generation
- EMI tracking using **Cron Jobs**

### 🧑‍💼 Admin Dashboard
Admins can:
- View and manage loan applications
- Add new borrowers
- Create and manage loan details
- Track and manage borrower payments

### 👤 Borrower Dashboard
Borrowers can:
- Securely view loan details
- Check EMI schedules and payment status

## Usage
1. Admin logs in and creates borrower profiles
2. Admin creates loan details and EMI schedules
3. Borrowers log in to view loan and EMI information
4. EMI schedules are automatically managed via cron jobs
